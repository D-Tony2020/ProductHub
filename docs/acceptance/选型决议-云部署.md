# ProductHub 云部署选型决议（P1）

> 方法：选型研究部 5 维并行 web 实证 + 逐维对抗复核。
> **数据质量声明（诚实）**：本环境对阿里云/腾讯云/华为云主域名的直连 WebFetch 被 socket 阻断，**"能力类"事实（PG16/PITR/高可用/备案规则）多为官方页实证，但具体"价格数字"多数为 WebSearch 摘取官方页/社区文章、未能逐字打开定价页核 → 标"待核"，落地前必须用各厂商价格计算器按目标地域复核**。AWS 价格经官方 Price List API 取得一手数据（2026-06-17）、DigitalOcean 官方定价页透明，可信度最高。

## 1. 约束回顾
规模保守（~8000 SKU / 并发 ≤20）｜RPO ≤15min｜**RTO 分钟级**｜SLA 务实档 99.5%｜整套月预算 **¥300-1000**（托管PG+主机+对象存储+流量）｜栈 FastAPI+PG16+Caddy+前端静态。

## 2. 能力结论（均官方实证，满足项目要求）
- **PG16 + PITR + 高可用**：阿里云 RDS PG、腾讯云 TencentDB PG、华为云 RDS PG、阿里云/腾讯云香港 RDS PG、AWS RDS PG、DigitalOcean Managed PG、Vultr Managed PG —— **全部支持 PG16、自动备份+WAL 归档 PITR、主备/多 AZ 高可用**，能力层面均可达 RPO≤15min、RTO 分钟级。
- **备案**：境内主机+境内域名 = 刚性需 ICP 备案（免费，3–22 工作日）+ 公安联网备案；香港/海外 = **免 ICP、免公安备案**。
- **跨境合规**：香港属"数据出境"，但据 2024《促进和规范数据跨境流动规定》，非关基、年向境外提供个人信息 <10 万人（非敏感）**豁免**安全评估/标准合同——本产品仅少量 B2B 联系人，远低阈值，**香港方案跨境合规近乎零额外审批**，仅需落实 PIPL 三件套（告知/最小必要/安全措施）。

## 3. 整套月成本对比（估算，价格待计算器核）
| 项 | 境内（阿里云生态，需备案） | 香港/海外（免备案） |
|---|---|---|
| 应用主机 2C4G | 阿里云 ECS u1 ≈ **¥17/月**（待核） | 阿里云香港轻量 ≈¥24/月起 / AWS Lightsail ≈¥168/月 |
| 托管 PG16 高可用 2C4G | 阿里云 RDS PG 高可用 ≈ **¥348/月**（+100GB 存储约+¥15；多源一致，待计算器核） | DigitalOcean 4G 单节点 ≈¥438/月；**HA standby 翻倍≈¥876**；AWS HK t4g.medium 单AZ≈¥637、**多AZ≈¥1274（超预算）** |
| 对象存储 100GB+备份 | OSS ≈ **¥15-30/月** | S3 ap-east-1 待核 / 可仍用境内 OSS |
| **整套（达 HA/分钟级 RTO）** | **≈ ¥400-450/月 ✓ 预算内** | HA 普遍 **¥900-1300，触顶/超预算** |
| **整套（单节点 PG，DB 无 HA）** | — | 阿里香港轻量+DO 4G ≈ **¥480/月**（RTO=恢复时长，非分钟级） |
| 上线前置 | ICP 备案 3-22 工作日 + 公安备案 | 无备案，最快 |
| 境内访问时延 | 低 | 较高（需 CN2 GIA 优化线路） |

## 4. 关键发现（决策支点）
**境内阿里云 RDS PG 高可用版仅 ¥348/月**，使"**分钟级 RTO 的全 HA**"恰好装进 ¥300-1000 预算（整套 ~¥400-450）；而**香港要达同等 HA 普遍超预算**，预算内只能上单节点 PG（PITR 有、但 DB 故障 RTO = 恢复时长，非分钟级）。即：**你定的"RTO 分钟级 + ¥300-1000"组合，境内能双达标且更省；香港需在"放宽 RTO"或"上探预算"之间取舍。**

## 5. 选型建议
- **首选：境内 · 阿里云生态**（ECS u1 2C4G + RDS PG16 高可用版 2C4G/100GB + OSS）。
  理由：唯一在 ¥300-1000 内**同时满足 RPO≤15min + RTO 分钟级 + 99.5%**，且最省（~¥400-450/月）；数据本地、合规最简；ECS 可平滑扩容。**代价 = 备案周期（3-22 工作日），需排进上线时间表。**
- **备选：香港 · 阿里云香港轻量 + 阿里云香港/DigitalOcean 托管 PG**。
  适用：若"免备案、最快上线、海外/多地访问"优先。需接受二选一：① 预算内用单节点 PG（DB 故障 RTO 放宽到恢复时长）；② 上 HA 则预算上探至 ¥900-1300。跨境合规已基本无障碍（<10万豁免）。

## 6. 落地前必做（无论选哪条）
1. 用厂商**价格计算器**按目标地域配置"高可用 2C4G + 100GB + 7天备份"，导出**精确月报价**替换本表估算。
2. 确认托管 PG 的 **PITR 保留期**（常见 7 天）与备份免费额度细则。
3. 境内：启动 **ICP 备案**（占用上线前置时间）；香港：选 **CN2 GIA** 优化线路降时延。
4. 据选定厂商，把对象存储/备份对接进上一轮灾备脚本（offsite-sync 的 rclone 远端）。
